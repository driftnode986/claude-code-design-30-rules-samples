# CLAUDE.md (myapp)

## このプロジェクトについて

myapp は Next.js + TypeScript の SaaS ダッシュボードです。

## 必ず守るルール

- git commit する前に **必ず** `npm run lint` を実行してください
- lint が失敗したら commit を中止してください
- lint を通してから commit してください

## コーディング規約

- 関数名は camelCase
- ファイル名は kebab-case

## テスト

- 変更したファイルに対応するテストを追加してください
- `npm test` でテスト実行
