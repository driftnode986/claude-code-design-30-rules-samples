# CLAUDE.md (myapp)

## このプロジェクトについて

myapp は Next.js + TypeScript の SaaS ダッシュボードです。

## 絶対に編集してはいけないファイル

以下のファイル/ディレクトリは **絶対に** 編集しないでください。
誤って編集すると、本番環境が壊れます。

- `.env`: 本番シークレット (DB 接続情報、API キー)
- `.env.production`: 同上
- `package-lock.json`: 依存関係のロックファイル (手動編集禁止)
- `.git/` 配下すべて: Git 内部状態
- `migrations/` 配下: DB マイグレーション (新規追加のみ可、既存編集禁止)

これらのファイルを変更したい時は、必ず人間に確認してから作業してください。

## コーディング規約

- 関数名は camelCase
- ファイル名は kebab-case
