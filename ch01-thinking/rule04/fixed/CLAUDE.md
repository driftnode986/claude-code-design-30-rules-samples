# PDF 要約スクリプト

PDF ファイルを読み込んで日本語の要約を生成します。

## 使い方

```bash
claude "@paper.pdf を読んで、1500字以内で日本語で要約して。
重要な数値と図表は箇条書きで残して。"
```

## 制約

- 1 ファイル単位で実行する
- 出力はそのままターミナルに流す
- 結果を保存したい場合は `> summary.md` でリダイレクトする

## 拡張する場合の判断軸

次のいずれかが出てきたら、対応するレイヤを 1 つだけ足す:

- 同じパターンを 3 回以上手で繰り返した → `.claude/skills/pdf-summary/`
  に手順を Skill 化する
- 複数 PDF を毎週バッチ処理したい → 単純な shell スクリプトに包む
  (Claude Code を呼ばずに既存ツールで完結することも検討する)
- 抽出結果と要約結果でコンテキストを分けたい → Subagent を 1 つ追加する
- 「PDF 以外を絶対に Read しない」セキュリティ境界が必要 → PreToolUse
  hook を 1 つ追加する

3 つすべてに該当しない限り、このスクリプトは 1 行プロンプトのまま動かす。
