agent が起動時に読む情報:

1. `pwd` でプロジェクトルートを確認
2. `read claude-progress.txt` でナラティブログを取得
3. `read feature_list.json` で `passes: false` の機能リストを取得
4. `git log --oneline -20` で直近の commit 粒度を把握
5. `git status` で未コミット変更を確認 (理想は clean)
6. `init.sh` を実行して dev server を起動

この 6 ステップで fresh context window でも prior session の続きから
作業を再開できる状態になります。
